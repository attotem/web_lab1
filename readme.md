# Лабораторна робота №1: Розробка статичного інтерфейсу Web-додатка

**Студент:** Шинкаренко Віталій  
**Група:** КВ-21

---

## Завдання до роботи

- Ознайомлення із засобами HTML5 та CSS3.
- Використання бібліотек, таких як Twitter Bootstrap/Material UI, для побудови адаптивного Web-інтерфейсу.
- Розробка статичних сторінок Web-додатка:
  - Реєстрація користувача (поля: ім’я, email, стать, дата народження)
  - Вхід до сайту (поля: email, пароль)
  - Профіль користувача (табличне відображення даних)
  - Сторінка "Про додаток" (емблема додатку, короткий опис)
  - Робоча сторінка додатка (наприклад, Телефонний довідник)
- Перевірка адаптивності сайту на різних пристроях (ПК, смартфон, планшет).
- Розгортання проєкту на GitHub Pages.
